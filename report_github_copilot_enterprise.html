⚠️ 风险清单

HIGH Code snippets transmitted to cloud for inference D3
Copilot sends code context (surrounding lines, open files) to GitHub/Microsoft cloud for generating suggestions. While Enterprise plan promises no training on customer data, the code is still transmitted to and temporarily processed in Microsoft's cloud infrastructure. This creates data exposure risk for organizations handling highly sensitive or classified code.
HIGH Limited prompt injection defense for suggestion engine D5
Copilot's suggestion engine lacks documented defenses against prompt injection via malicious code in workspace files or dependencies. An attacker who compromises a dependency or injects malicious comments in code could potentially influence Copilot's suggestions to include vulnerable patterns.
MEDIUM No on-premises deployment option D3
Copilot is a purely cloud-based SaaS offering with no on-premises or hybrid deployment option. Organizations requiring air-gapped development environments or strict data sovereignty cannot use Copilot while maintaining their security posture.
MEDIUM Limited audit granularity for code suggestions D4
Individual Copilot code suggestions are NOT logged for privacy reasons. Organizations requiring full audit trails of all AI-generated code contributions cannot track which specific suggestions were accepted, potentially creating compliance gaps in regulated industries.
LOW Extension security model still maturing D6
GitHub Copilot Extensions are a relatively new feature (2024+) with a security model that is still evolving. Extensions are not sandboxed in isolated processes and the signing/verification infrastructure is limited compared to mature plugin ecosystems.
