⚠️ 风险清单

HIGH Cascade agent executes commands without sandboxing D2
Windsurf's Cascade agent can execute terminal commands directly in the user's environment without sandboxing. A compromised dependency or malicious prompt injection could lead to arbitrary command execution on the developer's machine, potentially exfiltrating code or credentials.
HIGH Significant prompt injection attack surface D5
Cascade reads entire codebase contexts including comments, dependencies, and configuration files. Malicious content in any of these could influence the agent's behavior. No documented defense system against prompt injection attacks.
MEDIUM Code transmitted to cloud for inference D3
Windsurf sends code context to Codeium's cloud for AI inference. While enterprise plans offer hybrid deployment and promise no training on customer data, the core inference still requires cloud connectivity in standard configurations.
MEDIUM Limited audit logging and SIEM integration D4
Windsurf stores activity logs locally with no native SIEM integration. Organizations requiring comprehensive audit trails of all AI-assisted code changes cannot easily integrate Windsurf into their security monitoring infrastructure.
LOW MCP security model immature D6
MCP server support in Windsurf lacks whitelist/blacklist mechanisms, configuration signing, and OAuth support for remote servers. The MCP security model is still evolving and may pose supply chain risk.
